Dataset Analysis

Se analizarán los datasets "data2018.csv", "data2019.csv", "data2020.csv" y "datos202108.csv"

Campo Tipo Long Descripcion
Provincia String 2 Identificador numérico de la provincia
Municipio String 3 Identificador numérico del municipio
Estación String 3 Identificador numérico de la estación donde se toman medidas de las emisiones
Magnitud String 2 Identificador numérico del tipo de gas medido así como las unidades de la medición
Punto_Muestreo String 10 Identificador numérico del punto de muestreo, es la unión de los identificadores de provincia, municipio, estación y magnitud
Ano String 4 Año en el que se realizó la medición.
Mes String 2 Mes en el que se realizó la medición
Dx String 2 Valor obtenido en la medición el día x
Vx String 1 Validación del valor obtenido el día x

Naming strategy

  • Domain: http://spainaq.linkeddata.com/
  • Path: http://spainaq.linkeddata.com/aq/
  • Ontologies: http://spainaq.linkeddata.com/aq/ontologies/
  • Classes: http://spainaq.linkeddata.com/aq/ontologies/[ontologyName]#[className]
  • Properties: http://spainaq.linkeddata.com/aq/ontologies/[ontologyName]#[propertyName]
  • Resources: http://spainaq.linkeddata.com/aq/resources/
  • Individual: http://spainaq.linkeddata.com/aq/resources/[className]#[id]
